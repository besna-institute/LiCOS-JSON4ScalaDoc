    Package structure

    Notable packages includes:

    Protocol API
    • licos.protocol.engine and its sub-packages contain LiCOS's Protocol framework.
      • BOX is the container that manages information to give to analysis engines.
        • AuthBOX is the container that manages information to give analysis engines for authentication and authorization.
        • LobbyBOX is the container that manages information to give analysis engines for lobby.
        • VillageBOX is the container that manages information to give analysis engines for village.
      • licos.protocol.engine.processing contains LiCOS's protocol processing engines.
        • AuthProcessingEngine
        • LobbyProcessingEngine
        • VillageProcessingEngine
        • VillageProcessingEngine4Logger
      • licos.protocol.engine.analysis and its sub-packages contain LiCOS's protocol analysis engines.
        • licos.protocol.engine.analysis.auth contains LiCOS's protocol analysis engines for authentication and authorization such as AuthenticationAndAuthorizationRequestAnalysisEngine.
        • licos.protocol.engine.analysis.lobby contains LiCOS's protocol analysis engines for lobby such as AdvancedSearchAnalysisEngine.
        • licos.protocol.engine.analysis.village contains LiCOS's protocol analysis engines for village such as BoardAnalysisEngine.
    • licos.protocol.element.auth contains LiCOS's protocol entities for authentication and authorization such as AuthenticationAndAuthorizationRequestProtocol.
    • licos.protocol.element.lobby contains LiCOS's protocol entities for lobby such as AdvancedSearchProtocol.
    • licos.protocol.element.village contains LiCOS's protocol entities for village such as AvatarProtocol.
    Async Protocol API
    • licos.protocol.engine.async and its sub-packages contain LiCOS's Async Protocol framework.
    • licos.protocol.engine.async.processing contains LiCOS's async protocol processing engines.
      • AuthProcessingEngine
      • LobbyProcessingEngine
      • VillageProcessingEngine
      • VillageProcessingEngine4Logger
    • licos.protocol.engine.async.analysis and its sub-packages contain LiCOS's async protocol analysis engines.
      • licos.protocol.engine.async.analysis.auth contains LiCOS's async protocol analysis engines for authentication and authorization such as AuthenticationAndAuthorizationRequestAnalysisEngine.
      • licos.protocol.engine.async.analysis.lobby contains LiCOS's async protocol analysis engines for lobby such as AdvancedSearchAnalysisEngine.
      • licos.protocol.engine.async.analysis.village contains LiCOS's async protocol analysis engines for village such as BoardAnalysisEngine.
    JSON API (low-level API)
    • licos.json.engine and its sub-packages contain LiCOS's JSON framework.
      • BOX is the container that manages information to give to analysis engines.
      • licos.json.engine.processing contains LiCOS's JSON processing engines.
        • AuthProcessingEngine
        • LobbyProcessingEngine
        • VillageProcessingEngine
      • licos.json.engine.analysis and its sub-packages contain LiCOS's JSON analysis engines.
        • licos.json.engine.analysis.auth contains LiCOS's JSON analysis engines for authentication and authorization such as AuthenticationAndAuthorizationRequestAnalysisEngine.
        • licos.json.engine.analysis.lobby contains LiCOS's JSON analysis engines for lobby such as AdvancedSearchAnalysisEngine.
        • licos.json.engine.analysis.village contains LiCOS's JSON analysis engines for village such as BoardAnalysisEngine.
    • licos.json.element.auth contains LiCOS's JSON entities for authentication and authorization such as JsonAuthenticationAndAuthorizationRequest.
    • licos.json.element.lobby contains LiCOS's JSON entities for lobby such as JsonAdvancedSearch.
    • licos.json.element.village contains LiCOS's JSON entities for village such as JsonAvatar.

    val json: String = source.getLines().mkString("\n")
    source.close()
    json
  }

  import scala.concurrent.ExceptionContext.Implicits.global

  Json2VillageMessageProtocol.toProtocolOpt(aJSONExampleOfTheLiCOSProtocol, villageInfoFromLobby) match {
    case Some(protocol: VillageMessageProtocol) =>
      Await.ready(
        processingEngine
          .process(anExampleOfBOX, protocol)
          .map { messageProtocol: VillageMessageProtocol =>
            messageProtocol.toJsonOpt.foreach { json: JsValue =>
              System.err.println(Json.prettyPrint(json))
            }
          }
          .recover {
            case error: Throwable =>
              System.err.println(error.getMessage)
          },
        Duration.Inf
      )
    case _ =>
      System.err.println("No protocol")
  }
}
